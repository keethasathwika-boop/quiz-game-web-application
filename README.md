# Quiz Game Web Application

## 📌 Overview

The Quiz Game Web Application is an interactive and responsive web-based quiz developed using HTML, CSS, and JavaScript. It presents multiple-choice questions, provides instant feedback, tracks the user's score, and displays the final result. The project demonstrates dynamic DOM manipulation, event handling, and JavaScript logic in a clean and user-friendly interface.

---

## ✨ Features

- 🎯 Multiple-choice quiz questions
- ✅ Instant answer validation
- 📊 Real-time score tracking
- ⏭️ Next question navigation
- 🔄 Restart quiz functionality
- 📱 Fully responsive design
- 🎨 Clean and modern user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Launch `index.html` in your preferred web browser.

For the best experience, run the project using the **Live Server** extension in Visual Studio Code.
